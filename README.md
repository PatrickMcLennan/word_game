### Word Game

hangman-like game where users attempt to guess a word 1 letter at a time. incorrect guesses are tracked and users are scored based on who needed the fewest guesses and hints.

Next.js for a bare-bones SSR React app, JSON file for a DB.

### Running it

- **Dev**: `pnpm dev` and open `http://localhost:3000`
- **Game data**: See `data/state.example.json`, and make a `state.json` with each of your players.
